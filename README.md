# mist-drive

City of Vernonia ArcGIS JavaScript template application.

### Usage

This repository is setup as a template.

Create a new repository with this as a project.

```shell
npm install
```

### Development

```shell
npm start
```

### Scaffold a New Widget

```shell
npm run widget -- --name=<YourWidgetName>
```

### Build

```shell
npm run build
```

### Test

```shell
npm run test
```
